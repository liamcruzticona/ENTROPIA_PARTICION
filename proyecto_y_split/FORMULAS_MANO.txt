                    ENTROPIA DE SHANNON
              Calculo manual de H(x5)

  Formula:  H(X) = - SUM P(x) * log2 P(x)

  Variable x5 - valores: {0, 1, 2}

  Frecuencias:
    x5 = 0  -->  481 / 1000
    x5 = 1  -->  393 / 1000
    x5 = 2  -->  126 / 1000

  Probabilidades:
    P(x5=0) = 0.4810
    P(x5=1) = 0.3930
    P(x5=2) = 0.1260

  Paso 1 - Sustituir en la formula:
    H(x5) = -[ 0.4810 * log2(0.4810)
             + 0.3930 * log2(0.3930)
             + 0.1260 * log2(0.1260) ]

  Paso 2 - Calcular logaritmos base 2:
    log2(0.4810) = -1.0560
    log2(0.3930) = -1.3471
    log2(0.1260) = -2.9885

  Paso 3 - Multiplicar:
    0.4810 * (-1.0560) = -0.5079
    0.3930 * (-1.3471) = -0.5294
    0.1260 * (-2.9885) = -0.3766

  Paso 4 - Sumar los terminos:
    H(x5) = -[ (-0.5079) + (-0.5294) + (-0.3766) ]
    H(x5) = -[ -1.4139 ]

  Paso 5 - Resultado final:
    H(x5) = 1.4140 bits  [checkmark]

  H maxima para 3 valores: log2(3) = 1.5850 bits
  La distribucion no es uniforme (1.4140 < 1.5850)

     Razon = 0.3940 / (0.7870 * 0.4810) = 0.3940 / 0.3785 = 1.0408
